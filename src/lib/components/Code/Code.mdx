# Code

Used to display a snippet of code.

## Basic usage
By default the code snippet is displayed inline.
<Playground>
    <Code>
        lorem ipsum dolor
    </Code>
</Playground>

## Multiline
To display a block-level code snippet, use the **multiline** prop.
<Playground>
    <Code multiline >
        hello <br />
        world
    </Code>
</Playground>
