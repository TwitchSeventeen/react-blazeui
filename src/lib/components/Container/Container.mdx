# Container

Containers provide media query screen width restrictions on your content.

## Basic usage
```jsx
<Container size="xsmall"></Container>
<Container size="small"></Container>
<Container size="medium"></Container>
<Container size="large"></Container>
<Container size="xlarge"></Container>
<Container size="super"></Container>
```
