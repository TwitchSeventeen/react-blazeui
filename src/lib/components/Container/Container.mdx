# Container

Containers provide media query screen width restrictions on your content.

## Basic usage
```jsx
<Container size="xsmall"></Container> // 20em
<Container size="small"></Container> // 30em
<Container size="medium"></Container> // 48em
<Container size="large"></Container> // 64em
<Container size="xlarge"></Container> // 78em
<Container size="super"></Container> // 116em
```
