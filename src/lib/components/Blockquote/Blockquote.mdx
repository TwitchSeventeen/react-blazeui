# Blockquote

Use this to style a blockquote, optionally you can attribute the quote with a citation.

## Properties
<PropsTable of={Blockquote} />

## Basic usage
<Playground>
    <Blockquote>
        To quote use the following code to make it stand out from the rest of the page
    </Blockquote>
</Playground>

## Citation
Use the **cite** prop to add a footer citation to the quote.
<Playground>
    <Blockquote cite="~ Greg">
        To quote use the following code to make it stand out from the rest of the page
    </Blockquote>
</Playground>
