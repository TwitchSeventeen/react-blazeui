### Fixed Width
Use the **fixed** prop to fix the width of the field without it breaking the layout of the group.
<Playground>
    <Input.Group>
        <Input.Field>
            <Input />
        </Input.Field>
        <Input.Field fixed style={{ width: '50%' }}>
            <Input />
        </Input.Field>
        <Input.Field>
            <Input />
        </Input.Field>
    </Input.Group>
</Playground>
