# Card

Cards are a common minimalistic design tool used to segregate sections of your content.

## Header, Body and Footer
Use the header, body and footer components to construct some nice cards.

<Playground>
    <Card>
        <Card.Header>
            <h2 className="c-heading">
                Heading

            </h2>
        </Card.Header>
        <Card.Body>
            <p>Lorem ipsum dolor sit amet, feugiat corpora ex eam. Inciderint eloquentiam sea et.</p>
        </Card.Body>
        <Card.Footer>
            <div className="c-input-group">
                <button className="c-button c-button--block c-button--brand">Button</button>

            </div>
        </Card.Footer>
    </Card>
</Playground>

## Grouped Items
Grouping card items couldn't be easier. Add the **grouped** prop which will remove the item borders. Then insert a Card.Divider component into the list, which creates a horizontal line.
<Playground>
    <Card grouped>
        <Card.Item>Item 1</Card.Item>
        <Card.Item>Item 2</Card.Item>
        <Card.Divider separator />
        <Card.Item>Item 3</Card.Item>
        <Card.Item>Item 4</Card.Item>
        <Card.Item>Item 5</Card.Item>
        <Card.Divider separator />
        <Card.Item>Item 6</Card.Item>
        <Card.Item>Item 7</Card.Item>
        <Card.Item>Item 8</Card.Item>
    </Card>
</Playground>
