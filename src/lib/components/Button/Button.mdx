# Button

Probably the first thing people look at when choosing a UI toolkit. The common button!

## Button Colours 
Use the type prop to provide styling to the alert.

    <Button>Default</Button>

## Button Groups
To group related buttons together you can use the Button.Group element. This will bunch the buttons together and remove the spacing between them.
<Playground>
    <Button.Group>
        <Button type="brand">Button</Button>
        <Button>Button</Button>
    </Button.Group>
</Playground>

Adding the **rounded** prop will make the first and last buttons rounded.
<Playground>
    <Button.Group rounded>
        <Button type="brand">Button</Button>
        <Button>Button</Button>
        <Button type="info">Button</Button>
    </Button.Group>
</Playground>

## Button Type
By default buttons are rendered with a HTML type of _button_. Use the **buttonType** prop to change this to _submit_ or _reset_.
